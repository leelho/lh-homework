##  利用python解决高数问题  

### $\int \frac {3x}{(x^2+1)^2}dx$   

python公式如下  
``` 
   python 
   >>> import sympy 
   >>> integrate (x*3/(x**2+1)**2,x)
   -3/(2*x**2 + 2)
```  

###  $\lim\limits_{x\to0} \frac {\sqrt{x+1}-sqrt{1-x}}{x}$       

python公式如下  
```
   python
   >>> import sympy 
   >>> limit (( sqrt(1+x)-sqrt(1-x))/(x),x,0)  
   1
```
